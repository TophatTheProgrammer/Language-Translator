# Image-Text-Translator
a bad language translator that extracts its text from images.
My first ever real project made in python.
If you want to use it in games, you need to make it windowed fullscreen or just windowed.

Controls:
-hold alt and then click p to activate the screenshot state.
-click g two times to take the screenshot.
-screenshot will be saved as a .png image but you can stop it by removing the (TEMP!) line in textextractor.py

Requirements:
-keyboard
-threading
-time
-tkinter
-mss
-cv2
-numpy
-pytesseract (tesseract)
-pyautogui
-lingua
-deep_translator
